Help Here!
  • Before Changing Settings Please be sure that you know what you are doing as it can drastically change the functionality of the app
  • The Output Type option changes how you will see the results from the app: Traffic Lights will show you a different colour depending on the certainty of the program while Percentage will show a more accurate percentage of how certain the program is of its choices.
  • The Activation Limit option will change how certain the program needs to be before it returns a result.
  • The Output Log option will decode if you want to see the full out put of the program as it produces guesses for each class, essentially if Show All Outputs is selected the program will show how certain it is of each class rather than the ones it is most certain of, if this option is not selected only the most probable classes will be displayed
